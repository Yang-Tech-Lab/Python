## 📂 Key Projects

### 1. Web Automation & Data Extraction (Selenium)
A robust bot designed to automate search tasks and capture visual "Proof of Work" for clients.
- **Features:** Automated querying, headless browsing, and automated screenshot reporting.
- **Use Case:** Automating repetitive market research tasks on platforms like Fiverr.

### 2. Fiverr Sales & Financial Reporting (Pandas)
An automated data processing pipeline that transforms raw sales data into clean, client-ready Excel reports.
- **Features:** Automated revenue calculation, data cleaning, and professional formatting using Pandas.
- **Outcome:** Reduces manual reporting time by 90%.

### 3. PCB Design Workflow Automation
(Upcoming/Integrated) Python scripts designed to optimize KiCad 9.0 workflows, including automated BOM generation and impedance control checks.
